## 🌿 Overview

Pixel Garden Planner is a React-based productivity application that transforms task management into an interactive experience. Users complete daily and one-time tasks to earn Nature Points, unlock decorations, and gradually build a virtual pixel-inspired garden.

---

## ✨ Features

🌱 Create, edit, and delete tasks

📅 Support for Daily and One-Time tasks

✅ Mark tasks as completed

🍃 Earn Nature Points

🌼 Unlock pixel-inspired garden decorations

📊 Productivity dashboard

🔥 Daily streak tracking

💾 Local Storage persistence

📱 Responsive design

🌙 Dark mode support

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/pixel-garden-planner.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
